Winget

winget is a CLI package manager for Windows, which is similar to Linux. Finally, we are able to install software using the command line.

Note: For a smoother installation, please start cmd as admin. Press Win+R, type 'cmd' and before you hit return, hold CTRL+Shift while pressing return.

If you type winget in the command prompt and it says, it can not find it, winget is not installed then. In future versions of windows, it will come by default. Until then, download the latest release manually here!
